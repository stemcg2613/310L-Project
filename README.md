## Project Overview

This project is a PHP-based web application developed for the SDC310L course. The application demonstrates core web development concepts including server-side programming, database integration, MVC architecture, and application testing.

The application follows an online store concept where users can view products retrieved from a database and interact with the application through structured PHP pages and forms.

---

## Technologies Used

- PHP  
- MySQL  
- HTML / CSS  
- Apache (XAMPP)  
- Git & GitHub  

---

## Application Features

- Database-backed product data  
- PHP–MySQL database connection  
- CRUD operations (Create, Read, Update, Delete)  
- Shopping cart functionality  
- MVC (Model-View-Controller) architecture  
- Server-side form handling  
- Basic routing between controllers and views  

---

## Application Architecture

The application is structured using the MVC design pattern:

- **Models** handle database access and business logic  
- **Views** manage the user interface and presentation  
- **Controllers** process user requests and coordinate data flow  

This architecture improves code organization, maintainability, and scalability.
